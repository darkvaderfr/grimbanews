## GroundNews Key Features (To Replicate + Improve)

### 1. Bias Indicators (Core Feature)
- **Left/Center/Right bias badges** on every article
- **Blindspot detection** — stories only covered by one side
- **Media ownership transparency** — who owns each source
- **Funding sources** — corporate, state, independent

### 2. Article Comparison View
- **Same story, different angles** — side-by-side comparison
- **Headline comparison** — how each side frames it
- **Source diversity meter** — balance indicator

### 3. Clean, Readable Layout
- **Card-based article feed**
- **Clear typography hierarchy**
- **Minimal distractions** — focus on content
- **Quick actions** — save, share, compare

### 4. Personalization
- **Topic following** — select interests
- **Source filtering** — include/exclude sources
- **Reading history** — track consumption patterns
- **Bias balance score** — how balanced is your feed

### 5. Search + Discovery
- **Advanced search** — by topic, source, date, bias
- **Trending topics** — what's being covered
- **Underreported stories** — blindspot feed

## GrimbaNews Differentiators

### What Makes Us Better

1. **Francophone First** — Built for French-speaking Africa + diaspora
2. **AI-Powered Rewriting** — Same story, multiple perspectives (AI-generated)
3. **WhatsApp/Telegram Integration** — Daily digests via messaging apps
4. **Glass/Translucency Design** — Steve's cinematic design language
5. **Pan-African Focus** — African sources prioritized
6. **GroundNews API Alternative** — Open bias ratings for all sources
