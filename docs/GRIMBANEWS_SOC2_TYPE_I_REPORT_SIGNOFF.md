## Per-signoff procedure

1. Audit firm drafts report (T+1 month post-field-work).
2. Sara + Vader review draft.
3. Per-finding clarification feedback to audit firm.
4. Final report issued.
5. Vader formal signoff.
6. Per-customer distribution (per-customer NDA if needed).

## Per-customer distribution channels

- Per-customer secured portal.
- Per-customer per-NDA email.
- Per-public-trust-center upload (per Wave LLL `docs/GRIMBANEWS_TRANSPARENCY_REPORT_SCOPE.md`).

## Per-report metadata

- Report period (point-in-time for Type I).
- Per-criteria coverage (Security + Availability + others as applicable).
- Per-system in-scope.
- Per-finding count (clean vs with-findings).

## Per-renewal cadence

- Type I: annual.
- Type II: 6-12 months observation periods.
