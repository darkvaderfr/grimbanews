**Gating dependency:** Vader personally writes the reflection; cannot be generated.

## Why this exists

When Vader chooses to write a founder retrospective (5-year mark or sunset moment), prompts help structure the reflection. Saves blank-page anxiety.

## Prompt set

### What was the impulse?
- What did you see broken about news / media that made you start GrimbaNews?
- What would you tell yourself in Year 0 about Year N?
- What did you assume that turned out wrong?

### What were the breakthroughs?
- Which feature mattered most? Which one almost mattered nothing?
- Which editorial decision did you wrestle with longest?
- Which reader-moment validated the work?

### What were the regrets?
- What did you ship that shouldn't have shipped?
- What didn't you ship that you should have?
- Which hire / partnership / decision do you wish you'd done differently?

### What you'd carry forward?
- Which methodology insight transcends GrimbaNews?
- Which editorial framework deserves replication?
- Which technical decision was prescient?

### What this taught you about media + trust?
- Open-form section.

## Format suggestions

- 3000-5000 words.
- Per-section unstructured, follow energy.
- Publish at sunset or 5-year mark.
- Personal voice (vous → tu OK in founder voice).
- Archive in `docs/founder-retro/{year}.md`.
