# GrimbaNews — Contributor Program Launch Retrospective Scope

**Status:** plan v0
**Owner:** Liam Smith (PM) + Henry Walker (Editorial) + Lucy Leai (Strategy)
**Walks:** Mythos S1460 (contributor program launch retrospective) deferred → partial

## Why this exists

Without a post-launch retro, the program drifts. The retro forces honest assessment of intake-to-publish funnel, editor bandwidth, payout fairness, and reader response.

## Retro template

| Section | Lead |
|---|---|
| Funnel metrics | Liam — intake → submitted → published rates |
| Editor bandwidth | Henry — hours/submission, queue depth, SLA hit rate |
| Contributor satisfaction | Maria Lopez — NPS-style survey |
| Payout health | Warren Buffett / Ray Dalio — $/article, total spend, runway impact |
| Reader response | David Chen — read-rate vs in-house bylines |
| Editorial quality | Henry — corrections rate per-contributor |
| Process changes | Liam — concrete revisions for v1.1 |

## Output

- Internal exec doc circulated to NK + OM execs.
- Public summary (excluding individual contributor names) published as part of next annual transparency report (S2001).
- Backlog of v1.1 improvements drafted.
