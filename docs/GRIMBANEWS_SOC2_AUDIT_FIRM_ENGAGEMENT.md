## Shortlist (operator-side)

### Tier-1 (best for SaaS, $35-75K Type I)
- **A-LIGN:** common choice for SaaS, fast turnaround.
- **Schellman:** thorough, premium pricing.
- **Prescient Assurance:** mid-tier, good for fast-growth.

### Tier-2 (more accessible, $20-40K Type I)
- **Insight Assurance:** boutique, good service.
- **Linford & Co:** SaaS-focused.

### Tier-3 (smallest, $15-25K Type I)
- Regional firms; local CPA + AICPA-certified SOC 2 practice.

## Selection criteria

- AICPA membership + SOC 2 practice.
- SaaS / publisher industry experience.
- Per-engagement timeline (target: complete Type I in 4-6 months).
- Per-firm references + reviews.

## Per-engagement scope

- Type I: point-in-time assessment of controls (~$25-50K range).
- Type II: 6-month operational assessment (~$40-80K range; follow-on to Type I).

## Per-engagement timeline

- Month 0: engagement signed.
- Month 1-2: per-firm-led readiness assessment + remediation.
- Month 3-4: per-control evidence collection + interviews.
- Month 5: report drafted.
- Month 6: report finalized + delivered.
