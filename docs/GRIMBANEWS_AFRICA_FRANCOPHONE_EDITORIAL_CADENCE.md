## Why this exists

GrimbaNews is FR-primary. The Africa francophone audience (West Africa, Maghreb, DOM-TOM, diaspora) is a natural primary audience but coverage gap is real: most French sources are Hexagone-centric.

## Per-region cadence

- **West Africa core (Senegal, Côte d'Ivoire, Mali, Burkina Faso, Niger, Guinea):** Daily editorial pickup; morning + evening cycle aligned with UTC+0.
- **Central Africa (Cameroon, Gabon, Congo, DRC):** Daily pickup; align with UTC+1 evening cycle.
- **Maghreb (Morocco, Algeria, Tunisia):** Daily pickup; align with UTC+0 / UTC+1.
- **DOM-TOM (Réunion, Mayotte, Guadeloupe, Martinique, Guyane):** Per `docs/GRIMBANEWS_DOM_TOM_SOURCE_ROSTER.md`.
- **Diaspora content:** Sources from francophone press outside Africa covering African affairs (RFI, France 24, Le Monde Afrique, Africa Report).

## Source tier per region

- Tier-1: Established print + broadcast (Le Soleil SN, Fraternité Matin CI, El Watan DZ, etc.)
- Tier-2: Online native (Hespress MA, Tunisie Numérique TN, Africa Report)
- Tier-3: Partner newsrooms via partnership program

## Daily cadence

- 06:00 UTC: West + Central Africa morning sweep
- 12:00 UTC: Maghreb noon sweep
- 18:00 UTC: Africa evening + diaspora pickup
- Per-region cluster surfacing on /region/africa-francophone (per Wave AAFF localization)

## Editorial guardrails

- Ethnic / religious-conflict reporting: extra factuality check per Wave ZZZZ.
- Coup / unrest coverage: source balance critical (state-friendly + opposition).
- Per-cluster bias mix monitored daily.
