2. **Audit panel deliverables:**

3. **Vader written signoff:**
   - Personal note (paragraph minimum) acknowledging audit panel findings.
   - Per-finding: accept, defer, or reject.
   - Per-accepted: action plan + owner.
   - Per-rejected: rationale.

4. **Public signoff:**
   - `/transparence/mythos-closure` page published.
   - Per-stakeholder (reader / partner / customer) communication.
