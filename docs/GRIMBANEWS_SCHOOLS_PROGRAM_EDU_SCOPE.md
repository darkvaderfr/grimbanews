**Status:** plan v0
**Owner:** Lucy Leai (Strategy) + Steve Jobs (CPO) + per-region editor

**Gating dependency:** Per-region school-partnership program + curriculum-aligned methodology.

## Why this exists

GrimbaNews's bias-classified + Middle Ground / Blindspot signal is an ideal media-literacy teaching tool. Schools program offers free institutional access + classroom curriculum.

## Program tiers

- **Lycées (FR) / Gymnasium (DE) / Sec. Schools (UK):** free access for media-literacy classes.
- **Universities:** access + per-institution methodology workshops.
- **Adult-ed:** free MOOC-style modules.

## Classroom curriculum modules

- Module 1: What is media bias? (intro)
- Module 2: How GrimbaNews classifies sources (methodology)
- Module 3: Middle Ground vs Blindspot — editorial signals
- Module 4: Reading clusters side-by-side (workshop)
- Module 5: Building your own analysis (project)

## Per-region rollout

- FR: partnership with CLEMI (Centre for media literacy education).
- DE: with state Schulbehörde + KMK.
- Per-region partnerships via Wave LLL partnership program.
