**Status:** plan v0

## Per-International-edition cadence

- Daily 12:00 UTC (midpoint global timezone coverage).
- Weekly Sunday 10:00 UTC (deep editorial roundup).
- Per-major-event: per-summit / per-crisis specials.

## Per-International template

- Per-region hero (Europe / Americas / Asia-Pacific / MENA / Sub-Saharan Africa).
- Per-region cluster picks.
- Per-International Middle Ground signal (where global perspectives converge).
- Per-International Blindspots.
- Per-International investigative spotlight.
- International curator's pick + commentary.

## Per-edition subscription tier

- Free: weekly only.
- Premium: daily + weekly + per-event.

## Per-locale variants

- English (default).
- French (per-edition translation via Wave SUB-27 translator pool for high-stakes pieces).
