# GrimbaNews — Adult-Ed Citizenship Prep Plan

**Status:** plan v0
**Owner:** Henry Walker (Editorial) + Lucy Leai (Strategy) + Maria Lopez (Community)
**Walks:** Mythos S2125 (adult-ed program — citizenship-prep partnership) deferred → partial
**Gating dependency:** S2121-S2124 adult-ed program scaffolding + per-jurisdiction citizenship-test syllabus mapping.

## Why this exists

Newcomers preparing for citizenship tests (FR naturalisation, Canada citizenship test, US USCIS) need reliable, bias-balanced, plain-language news to build civic literacy. The intersection of Grimba's mission + adult-ed program creates a natural curriculum module.

## v1 jurisdiction scope

- France: Livret du citoyen + Charte des droits et devoirs du citoyen français.
- Canada: "Discover Canada" guide.
- US: USCIS 100-civic-questions + reading list.

## Module shape

- Per-week reading-list (5-7 articles) mapped to test topics.
- Plain-language vocab gloss for civic terms.
- Quiz mode (mini-quizzes per topic).
- Test-day prep checklist.

## Partner format

- Library partnerships (BAnQ, NYPL, public library networks).
- Community-center partnerships (YMCA, Maison de quartier).
- Free for partner-enrolled learners; partner consents to anonymous aggregate analytics.

## Anti-patterns

- No political-leaning bias in content selection (must reflect official curriculum, not editorial position).
- No paid-tier gating on citizenship-prep content.
- No data-sale of learner reading habits.
