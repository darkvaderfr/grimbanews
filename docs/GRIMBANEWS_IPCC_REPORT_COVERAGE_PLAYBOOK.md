**Status:** plan v0
**Owner:** Lucy Leai (Strategy) + climate editor TBD

**Gating dependency:** Climate editor + per-IPCC-cycle calendar (next AR7 expected 2028-2029).

## Why this exists

IPCC reports (AR cycles, Special Reports) are landmark editorial events. Mainstream coverage is uneven: science correctly summarized in ~10% of articles, politicized framing in the rest. GrimbaNews plays both honest-science + comparative-political-framing roles.

## Per-IPCC-event arc

T-2 weeks: Pre-publication source curation (which expert authors are reachable for comment).
T-1 week: Pre-publication framing primer (what the report is and isn't).
T-0 (publication day): Live coverage cluster with multi-source side-by-side.
T+24h: Settled-narrative cluster with science-vs-policy explainer cross-link.
T+1 week: Reception analysis (how each political camp framed it).
T+1 month: Implementation tracker setup.

## Per-report dossier structure

- Hero cluster: the report's headline finding.
- Sub-cluster per chapter (mitigation / adaptation / regional / cross-cutting).
- "Comment on classe les claims scientifiques" methodology cross-link.
- IPCC author quotes panel (per Wave UUUU cluster-quote-extraction).

## Editorial transparency

Per-IPCC-event published "comment on a couvert ce rapport" reader-facing post-mortem.
