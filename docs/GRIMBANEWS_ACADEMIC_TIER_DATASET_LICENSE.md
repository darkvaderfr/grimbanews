## Why this exists

Academic users need clear license terms permitting research use + reproducibility + grant-funded analysis without negotiating per-user.

## Tier-specific license

### Academic free tier
- License: CC-BY-4.0
- Permitted: research, teaching, peer-review reproduction
- Required: attribution per Wave SUB-25 citation format
- Restricted: commercial-resale prohibited

### Academic with grant tier (free, gated)
- Same as above plus:
- Per-grant data export accelerated
- Per-grant DOI assigned
- Per-grant dedicated email contact

### Commercial academic tier (paid, future)
- CC-BY-NC adapted (non-commercial)
- Bulk data export
- Per-customer SLA

## Per-tier verification

- Academic free: self-declaration + .edu / institution email.
- Academic with grant: grant-ID verification.
- Commercial academic: invoice + counsel review.

## Per-tier renewal cadence

12 months (per Wave SUB-26 sister doc S1699).
