**Status:** plan v0

**Gating dependency:** Overlaps S2148 IFCN signatory pursuit; this doc captures the per-investigation workflow specifically.

## Why this exists

Investigative pieces are high-claim-density + high-stakes. Standard editor review insufficient; per-claim verification needed.

## Per-claim verification workflow

For each investigative piece:

1. **Claim extraction:** lead investigator extracts every factual claim.
2. **Per-claim source citation:** each claim mapped to primary source (document, interview transcript, dataset row).
3. **Per-claim verifier review:** independent reviewer (not co-author) confirms claim matches source.
4. **Per-claim grading:** "verified" / "supported-with-caveat" / "single-source" / "removed".
5. **Pre-publish:** counsel review of all "supported-with-caveat" + "single-source" claims.
6. **Per-piece fact-check sign-off log** archived.

## Verifier role

- Not co-author of piece (independence requirement).
- 5+ years journalism / research experience.
- Per-piece time: 30-60 hours.
- Can be in-house or contracted per piece.

## Per-piece public-facing fact-check note

Published alongside investigation: "This investigation contains X verified claims, Y supported-with-caveat, Z single-source. Each claim cites its primary source."
