# GrimbaNews — Long-Form Investigations Scope

**Status:** plan v0
**Owner:** Lucy Leai (Strategy) + investigations editor TBD
**Walks:** Mythos S2201 (Long-form investigations scope decision + first-investigation pick) deferred → partial
**Gating dependency:** Investigations editor + per-investigation budget.

## Why this exists

Beyond aggregation, GrimbaNews can host original long-form investigations on topics underserved by mainstream press. Examples:
- Per-source ownership chain analysis (who really owns what newspaper)
- Per-region undercover-of-power coverage
- Per-cluster narrative-deconstruction
- Per-event timeline-reconstruction

## First-investigation pick candidates

Per Lucy's strategy:
1. "Qui contrôle vraiment la presse française? — étude des propriétés croisées" — ownership-graph deep-dive.
2. "Le silence sur X" — blindspot analysis with editor commentary.
3. "Comment se forme un consensus médiatique" — Middle Ground signal explainer with case studies.
4. "L'aggrégation et la concentration: que se passe-t-il quand on perd la diversité" — meta-analysis.

## Per-investigation structure

- 10,000+ word piece (or multi-part series)
- Investigative-team byline
- Data + sources fully disclosed
- 30-day pre-publish counsel review
- Multi-format output: web + PDF + audio
- Per-investigation ombudsman pre-review (per Wave LLL)

## Cadence

- 1 long-form investigation / quarter (steady state).
- Per-investigation pre-launch promo cadence.

## Budget

Per-investigation: investigator time + counsel review + production. Operator estimate $5-20K per piece.
