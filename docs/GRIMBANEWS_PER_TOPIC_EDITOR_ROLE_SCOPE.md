# GrimbaNews — Per-Topic Editor Role Scope

**Status:** plan v0
**Owner:** Steve Jobs (CPO) + Lucy Leai (Strategy)
**Walks:** Mythos S1035 (per-topic editor roles) deferred → partial
**Gating dependency:** Per-topic hiring budget + editorial workflow v2 (S1291-S1300 partial from Wave LLL).

## Role

One editor per active v2 topic (40 max per `docs/GRIMBANEWS_TOPIC_TAXONOMY_V2_DESIGN.md`). Editor owns:

1. **Editorial brief** for the topic (signed off + maintained).
2. **Per-cluster bias placement review** — flags clusters where the auto-classifier looks off.
3. **Source-roster gap monitoring** — pages Lucy when bias mix drifts beyond ±5%.
4. **Per-topic newsletter** curator's pick (per `docs/GRIMBANEWS_PER_TOPIC_NEWSLETTER_DESIGN.md`).
5. **Reader-feedback triage** for the topic's surface.

## Time commitment

- Soft launch: 5 hours/week per topic.
- Steady state: 10 hours/week per topic.
- High-traffic topics (Politics, Climat during major events): up to 20 hours/week.

## Hiring path

- Internal first: Lucy Leai's strategy team for first 5 topics.
- External pickup: freelance editors via Iboga's editorial network for remaining 35.
- Tier-1 topics (Politics, Économie, International) get a dedicated full-time hire.
