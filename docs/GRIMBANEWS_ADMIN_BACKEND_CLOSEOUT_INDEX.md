## Current Backend Contract

- Custom Grimba admin surfaces use the shared cinematic shell.
- Dropdowns are solid and stacked above page actions.
- Dark/light mode parity is covered by CSS markers and route render tests.
- Core admin queues have responsive table cards and explicit inline actions.
- Source, RSS, story, cookie, and provider settings forms use shared form sections.
- NobuAI remains the public AI brand; provider names stay inside admin-only views.
- Deployment is intentionally deferred until the user explicitly requests production deployment.
