# GrimbaNews — Contributor Case Studies Scope

**Status:** plan v0
**Owner:** Henry Walker (Editorial) + Maria Lopez (Community)
**Walks:** Mythos S1459 (contributor case studies) deferred → partial
**Gating dependency:** ≥ 1 real contributor with ≥ 3 published pieces.

## Why this exists

Public case studies of contributor journeys (intake → publish → reach) help recruit the next wave and signal that the program is real. Without one real contributor, this is fiction; surrogate is the contributor program page itself.

## v1 case-study template

Each case study:

1. Contributor profile (name, locale, beat, day-job, languages).
2. Submission journey (timeline, revisions, editor partnership).
3. Published article(s) with attribution.
4. Reach (unique readers, time-on-page, partner pickup count).
5. Contributor's own takeaways (interview Q&A).
6. Program improvements suggested.

## Cadence

- 1 case study per quarter once 3+ contributors are active.
- Published at `/auteurs/parcours/{slug}`.
- Linked from contributor program landing.

## Privacy

- Contributor consent in writing.
- Numeric reach figures aggregate-only (not per-article exact unless contributor approves).
- Per-jurisdiction privacy law check before publication.
