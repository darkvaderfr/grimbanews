## Cadence per locale

| Locale | Daily | Weekly | Topical | Special-event |
|---|---|---|---|---|
| FR | ✓ 07:00 UTC | Sat 09:00 | ✓ | per-event |
| EN | ✓ 12:00 UTC | Sat 14:00 | ✓ | per-event |

| ES | ✓ 08:00 UTC+1 | Sat 09:00 | ✓ | per-elecciones |
| Other | weekly only at launch | + ramp up |

## Per-cadence editor responsibilities

- Daily: 30-min review of NobuAI-generated draft prior to send.
- Weekly: 60-min curation of standout stories.
- Topical: editor curates from cluster pool.
- Special-event: dedicated launch process.

## Subscriber acquisition flow

- Onboarding offers locale-default cadence subscription.
- Account-settings page allows per-cadence toggle.
- Per-cadence preview before subscribing.
